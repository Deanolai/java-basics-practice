# Java Basics Practice

Java SE 核心知识点巩固练习项目，包含三个独立业务子系统：

| 项目 | 说明 | 核心知识点 |
|------|------|-----------|
| 📚 library-management | 图书借阅管理系统 | 封装 · ArrayList · 异常处理 · String |
| 💰 payroll-system | 员工薪资管理系统 | 继承 · 接口 · 多态 · HashMap |
| 🪪 campus-card-system | 校园卡管理系统 | abstract · static · package · 综合 |

---

## 项目结构

```
├── requirements/               ← 任务清单与源文件列表
├── library-management/         ← Project 1
├── payroll-system/             ← Project 2
├── campus-card-system/         ← Project 3
└── README.md
```

详见 `requirements/tasks.md` 获取完整任务描述及进度追踪。
